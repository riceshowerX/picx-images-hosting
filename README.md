# 欢迎使用 PicX-riceshowerX 图床仓库

[PicX](https://github.com/XPoet/picx) 是一个简单而强大的图片托管工具。它支持通过 GitHub 仓库进行图片托管服务。

PicX 是完全开源的，您可以免费使用它。

如果您喜欢它，请在 [GitHub](https://github.com/XPoet/picx) 上给它点个星。

此仓库是我个人的 PicX 图床仓库。您也可以访问我的个人网站 [miksz.cc](https://miksz.cc) 了解更多信息。

# 食用方法
- GitHub Pages（默认）：`https://{{owner}}.github.io/{{repo}}/{{path}}`
- GitHub：`https://github.com/{{owner}}/{{repo}}/raw/{{branch}}/{{path}}`
- jsDelivr：`https://cdn.jsdelivr.net/gh/{{owner}}/{{repo}}@{{branch}}/{{path}}`
- Statically：`https://cdn.statically.io/gh/{{owner}}/{{repo}}@{{branch}}/{{path}}`
- ChinaJsDelivr：`https://jsd.cdn.zzko.cn/gh/{{owner}}/{{repo}}@{{branch}}/{{path}}`
